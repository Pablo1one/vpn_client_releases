# LightningMcQueen VPN - Windows

Впн-клиент для windows 10/11 (x64). Прото: vless (reality/ws/grpc/xhttp), tuic,
hysteria2, amneziawg, каскад через cloudflare warp. Маршрутизация «весь трафик» /
«Россия напрямую», split-tunnel, блокировка рекламы.

## Установка
1. Скачайте `LightningMcQueen-Setup.exe` из последнего релиза:
   https://github.com/Pablo1one/vpn_client_releases/releases/latest
2. Запустите установщик (нужны права админа - для tun-адаптера).

Обновления - внутри приложения. Исходный код приватный.
